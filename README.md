# Password Protection

## Description

Worry less and accomplish more with this all-in-one password management app. It includes everything you need to manage your passwords: Save passwords, view saved passwords, search among saved passwords, and delete saved passwords. ## Getting Started

### Dependencies

* MacOS Big Sur
* Windows 10

### Installing

* Clone with git
```
$ git clone https://github.com/Periyot/Password-Protection.git
```

* Modify the data path in DataManager.cs
```csharp
private string cs = @"URI=file:PATH"; // Path = Data path
```
